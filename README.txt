This is a reposity of code used for population genetic analyses of the spoon-billed sandpiper and red-necked stint genomes. 

00_Data directory contains gene annotaion file. Other files that should be added to this directory include refenrece genome (NCBI assembly: ASM369795v1) and bam files, that are based on the raw sequencing reads (NCBI Bioproject: PRJNA419629 ) mapped to the reference genome.
